**Toggle area (D-16):**
- Render one `<Toggle>` per `orgMember` in a `<div className="flex flex-wrap gap-2">`
- Each Toggle: `pressed` = member is in the current split assignees
- Toggle content: `<Avatar size="sm" src={member.avatarUrl} fallback={member.displayName[0]} />` + `<span className="ml-1.5 text-sm">{member.displayName}</span>`
- Toggle sizing: `className="h-9 min-w-[120px] px-3"`
- Toggle pressed state override (applied at usage — not in base component):
  `data-[state=on]:bg-primary/10 data-[state=on]:text-primary data-[state=on]:border-primary/30`
- On toggle press: add member to assignees array, recalculate LRM even split across ALL pressed members using `lrmSplit(totalAmountCents, pressedCount)`
- On toggle un-press: remove member from assignees array, recalculate LRM even split across remaining pressed members

**Create mode defaults (D-17):**
- 1 org member → auto-pressed (100% split)
- 2+ org members → all un-pressed

**Edit mode initialization (D-18):**
- `transaction.assignees.map(a => a.memberId)` → set these as initially pressed
- If pressed members have non-even distribution → open Customize split Collapsible initially

**"Customize split" Collapsible (D-16):**
- Only render when at least one toggle is pressed
- `<Collapsible>` wrapping a `<CollapsibleTrigger>` text button and `<CollapsibleContent>`
- Trigger label: "Customize split" with `ChevronDown`/`ChevronUp` (16px) icon after the text
- Trigger: `<Text variant="body-sm" className="text-muted-foreground hover:text-foreground cursor-pointer flex items-center gap-1">`
- Collapsed by default in create mode; open by default in edit mode when distribution is non-even
- `<CollapsibleContent>` renders `<SplitSection form={form} orgMembers={pressedMembers} transaction={transaction} />`
- Only pass `pressedMembers` (filtered from orgMembers) to SplitSection so it only shows rows for pressed members

**State management:**
- `pressedMemberIds` as local state (string array)
- `isCustomizeOpen` as local state (boolean)
- When toggle is pressed/unpressed: update pressedMemberIds, recalculate even split, sync to form assignees array via `form.setFieldValue('assignees', newAssignees)`
- LRM even split: `lrmSplit(form.getFieldValue('amount') * 100, pressedMemberIds.length)` — amount is dollars in form state, convert to cents

**React 19 note:** No `forwardRef` — pass `ref` as regular prop if needed (vercel-composition-patterns react19-no-forwardref rule).
