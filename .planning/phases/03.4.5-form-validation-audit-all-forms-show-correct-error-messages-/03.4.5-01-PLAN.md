```tsx
function FieldError({
  className,
  children,
  errors,
  ...props
}: React.ComponentProps<'div'> & {
  errors?: ({ message?: string } | undefined)[];
})
```
Internally: extracts `error.message` per entry, deduplicates via Map, renders:
- Single error → `<div role="alert" data-slot="field-error" className="text-sm font-normal text-destructive">message</div>`
- Multiple errors → `<div role="alert" ...><ul className="ml-4 flex list-disc flex-col gap-1"><li>…</li></ul></div>`
- Empty / no errors → returns `null`
