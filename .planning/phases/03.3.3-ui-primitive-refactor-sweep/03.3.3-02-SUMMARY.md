# Phase 03.3.3 Plan 02: Heading migration across 9 apps/web files Summary

**All raw `h1`–`h6` heading elements replaced with `<Text as="hN" variant="...">` in 9 apps/web component files, establishing Text as the single heading primitive**

- **Started:** 2026-04-17T00:00:00Z
- **Completed:** 2026-04-17T00:03:00Z
- **Tasks:** 2

## Accomplishments

**Task 1 — Settings components (5 files):**
- `HouseholdOverviewSection.tsx`: `<h2 className="font-heading text-xl font-semibold">` → `<Text as="h2" variant="h3">`
- `MembersSection.tsx`: `<h2 className="font-heading text-xl font-semibold">` → `<Text as="h2" variant="h3">`
- `CategoriesSettings.tsx`: h1 → `<Text as="h1" variant="h3">`; two h2 sub-section labels → `<Text as="h2" variant="label" className="font-semibold">`
- `MerchantRulesSettings.tsx`: `<h1 className="font-heading text-xl font-semibold">` → `<Text as="h1" variant="h3">`
- `_layout.settings/route.tsx`: `<h1 className="text-xl font-semibold">` → `<Text as="h1" variant="h3">` (h3 variant adds font-heading per D-05)

**Task 2 — Page-level components (4 files):**
- `Dashboard.tsx`: h1 → `<Text as="h1" variant="h3" className="mb-4">` (mb-4 layout class preserved)
- `Transactions.tsx`: h1 → `<Text as="h1" variant="h3" className="min-w-0 truncate">` (layout classes preserved)
- `Accounts.tsx`: h1 → `<Text as="h1" variant="h3" className="min-w-0 truncate">` (layout classes preserved)
- `Onboarding.tsx`: h1 (text-3xl) → `<Text as="h1" variant="h1">` (h1 variant = font-heading text-3xl font-semibold tracking-tight leading-tight)

All 9 files now import `Text` from `@ploutizo/ui/components/text`.
