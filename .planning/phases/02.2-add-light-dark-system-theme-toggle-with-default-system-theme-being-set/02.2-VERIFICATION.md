### Deviations from Plan (Approved)

Two intentional deviations from the plan's task specification were made and approved by the user:

1. **Icon change:** Plan specified `Monitor` (MonitorIcon) for system mode. Implementation uses `SunMoon` (SunMoonIcon). Better visual language — half-sun/half-moon matches the Sun/Moon pair more directly. User approved at checkpoint.

This is the shadcn recommended pattern and means all Tooltip usage across the app shares one provider context. User requested this during checkpoint.

Both deviations improve the implementation; neither affects the observable truths or the goal.
