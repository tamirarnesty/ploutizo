## Summary

Installed `next-themes`, wired ThemeProvider into the app root, and added a cycling ThemeToggle icon button to the sidebar footer. System theme is the default; preference persists via localStorage.

## What Was Built

- `next-themes ^0.4.6` added to `packages/ui`
- `ThemeProvider` re-exported from `@ploutizo/ui/components/theme-provider` — wraps app root with `attribute="class"`, `defaultTheme="system"`, `enableSystem`, `storageKey="theme"`
- `ThemeToggle` cycling button: SunMoon → Sun → Moon → SunMoon with tooltip ("System" / "Light" / "Dark")
- `useTheme` re-exported from `@ploutizo/ui/hooks/use-theme`
- `TooltipProvider` (delayDuration=500) hoisted to app root so all components can use `<Tooltip>` without a local provider
- `suppressHydrationWarning` added to `<html>` (next-themes requirement)
