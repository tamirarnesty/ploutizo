## Visual Treatment

| Option | Description | Selected |
|--------|-------------|----------|
| Minimal — text only | Centered like Onboarding: heading + subtext + button. No icons. | |
| Light icon — muted, inline | Small Lucide icon inline before heading. | |
| **Terminal/dot-art aesthetic** | Braille/dot-art "404" centerpiece, status bar, `ERROR · 0x404` label | ✓ |

**User's choice:** Terminal/dot-art aesthetic — inspired by reference image showing braille-style "404" dot art, top status bar (`● STATUS — 404`), `ERROR · 0x404` label, friendly heading, action buttons at bottom.

**Notes:** User shared reference screenshot. Said: "I want the 404 page to be interesting, not just minimal. It should have character, with ascii or braille style art, which is trending right now." They specified to ignore the search bar and exact button design, but adopt the overall aesthetic.
