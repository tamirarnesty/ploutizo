## Decisions Made

- `import.meta.env.DEV` gates all dev-mode error detail — Vite tree-shakes this block from production bundles, no stack traces leak to users
- Status bar path strings are static constants, not derived from `window.location` — no user input is ever rendered on these pages
- `error.message` and `error.stack` use JSX string interpolation only — React escapes content automatically
