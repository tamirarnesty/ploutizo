# Summary: Build Text typography primitive

## What Was Built

Created `packages/ui/src/components/text.tsx` — the single typography primitive for the app.

## Implementation

- **Polymorphic component:** `<Text as variant className>` with `as` (DOM element) and `variant` (visual preset) fully decoupled
- **10 allowed elements:** h1–h6, p, span, div, label — defaults to `p`
- **7 visual variants:** h1, h2, h3 (font-heading/Noto Serif), body, body-sm, caption, label (Geist/system font) — defaults to `body`
- **Plain Record lookup:** `variantClasses: Record<TextVariant, string>` — no cva needed for single dimension
- **React 19:** `ref` as regular prop, no `forwardRef`
- **className merged last** via `cn()` — call-site overrides always win
- **No "use client"** — component library; consumers decide

## Exported API

```ts
export type TextAs = "h1" | "h2" | "h3" | "h4" | "h5" | "h6" | "p" | "span" | "div" | "label"
export type TextVariant = "h1" | "h2" | "h3" | "body" | "body-sm" | "caption" | "label"
export { Text }
```

Consumed as `@ploutizo/ui/components/text` via existing wildcard path export — no package.json changes required.
