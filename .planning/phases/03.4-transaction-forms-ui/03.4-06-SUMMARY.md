# Phase 03.4 Plan 06: Select Label Display Fix Summary

All 8 Select fields in the transaction form now render human-readable labels instead of raw enum values or UUIDs — resolved by explicit `SelectValue` children using label maps and array `.find()` lookups.

## Tasks Completed

| Task | Name | Commit | Files |
|------|------|--------|-------|
| 1 | Fix type, incomeType, investmentType SelectValue (enum label maps) | ad99d5d | TransactionTypeFields.tsx, IncomeFields.tsx, ContributionFields.tsx |
| 2 | Fix accountId, categoryId, toAccountId, settledAccountId SelectValue (array lookup) | 6f2baab | TransactionForm.tsx, ExpenseFields.tsx, RefundFields.tsx, TransferFields.tsx, SettlementFields.tsx |

## Root Cause

Base UI `Select.Value` renders the raw `value` prop when no children are supplied — it cannot auto-resolve labels from `SelectItem` children because those items mount in a Portal that only appears when the popover is open. Fix: pass explicit children that compute the label inline.

## Changes

**Task 1 — Enum label maps (3 files):**
- `TransactionTypeFields.tsx`: Added `TYPE_LABELS` const at file scope; replaced `<SelectValue />` (self-closing) with `<SelectValue>{TYPE_LABELS[field.state.value] ?? 'Select type'}</SelectValue>`
- `IncomeFields.tsx`: Added `INCOME_TYPE_LABELS` const; replaced `<SelectValue placeholder="Select type" />` with children form
- `ContributionFields.tsx`: Added `INVESTMENT_TYPE_LABELS` const; replaced `<SelectValue placeholder="Select type" />` with children form

**Task 2 — Array-backed selects (5 files):**
- `TransactionForm.tsx`: `accounts.find((a) => a.id === field.state.value)?.name ?? 'Select account'`
- `ExpenseFields.tsx`: `categories.find((c) => c.id === field.state.value)?.name ?? 'Select category'`
- `RefundFields.tsx`: `categories.find((c) => c.id === field.state.value)?.name ?? 'Select category'`
- `TransferFields.tsx`: `accounts.find((a) => a.id === field.state.value)?.name ?? 'Select account'`
- `SettlementFields.tsx`: `accounts.find((a) => a.id === field.state.value)?.name ?? 'Select account (optional)'`
