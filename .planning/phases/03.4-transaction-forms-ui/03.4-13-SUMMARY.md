# Phase 03.4 Plan 13: Row Action Button Hover Visibility Summary

Row action button (MoreHorizontal) is invisible at rest and appears only when the `<tr>` ancestor is hovered, the dropdown is open, or keyboard focus is applied.

## What Was Built

Single className change on the `<Button>` inside `<DropdownMenuTrigger>` in the actions column cell renderer:

- **Before:** `"text-muted-foreground hover:text-foreground"` — button always visible, muted color
- **After:** `"opacity-0 [tr:hover_&]:opacity-100 data-[state=open]:opacity-100 focus-visible:opacity-100"` — button hidden at rest, revealed on row hover

The `[tr:hover_&]` Tailwind v4 arbitrary parent-selector variant targets the nearest `<tr>` ancestor's `:hover` state, making this a pure CSS solution with no JS event handling.
