## What Was Built

### Task 1 ‚Äî FormattedAmountInput

New file: `apps/web/src/components/transactions/FormattedAmountInput.tsx`

A controlled text input that:
- Shows Intl.NumberFormat-formatted value (e.g. `1,000.00`) when not focused
- Reverts to raw number string on focus for editing
- Uses `useRef` focused guard to prevent the `useEffect` sync from overwriting user input while typing
- Uses `inputMode="decimal"` for mobile numeric keyboard without `type="number"` browser quirks
- Wraps InputGroup/InputGroupAddon/InputGroupText/InputGroupInput from `@ploutizo/ui/components/input-group` (all exports verified before use)

### Task 2 ‚Äî TransactionForm three fixes

**B. Calendar auto-close** ‚Äî Added `const [dateOpen, setDateOpen] = useState(false)`. Changed `<Popover>` to controlled (`open={dateOpen} onOpenChange={setDateOpen}`). Added `setDateOpen(false)` in Calendar `onSelect` callback.
