## What Was Built

**Task 1 — Amount type change:**
- `types.ts`: `amount: number | undefined` (was `number`)
- `buildDefaultValues` create branch: `amount: undefined` (was `amount: 0`) — amount field blank on create
- `toApiPayload`: `Math.round((value.amount ?? 0) * 100)` guards undefined

**Task 2 — Amount InputGroup + Calendar date picker:**
- Amount field replaced with `InputGroup > InputGroupAddon("$") + InputGroupInput` compound
- Amount onChange: `e.target.value === '' ? undefined : parseFloat(e.target.value)`
- Amount validator: `!value || value <= 0` handles undefined
- Date field replaced with Base UI `Popover > PopoverTrigger(render prop) + Calendar`
- Date shows formatted string (`MMM d, yyyy`) or "Pick a date" in muted-foreground
- Date stores YYYY-MM-DD string via `date-fns format()`
- IIFE wraps Calendar block to scope `selectedDate` without polluting AppField render scope

**Task 3 — Icon-only Delete button:**
- `size="icon"` added to Button inside AlertDialogTrigger
- `aria-label="Delete transaction"` for accessibility (WCAG icon-only requirement)
- Removed "Delete transaction" text node from AlertDialogTrigger children
- Removed `className="mr-1"` from Trash2 (no label to space from)
- Added `hover:bg-destructive/10` for hover visual feedback
