## What Was Built

### useThemeKeyboardShortcut hook

New hook at `apps/web/src/hooks/useThemeKeyboardShortcut.ts` that:

- Listens for `keydown` events on `window`
- Only acts on `e.key === "d"` with no modifier keys (meta/ctrl/alt/shift)
- Skips when focus is in INPUT, TEXTAREA, SELECT, or contenteditable elements
- Cycles theme: system → light → dark → system via next-themes `setTheme`
- Uses the same `cycleMap` pattern as `ThemeToggle` in packages/ui
- Implements `advanced-event-handler-refs` pattern: handler stored in `useRef`, effect runs once with `[]` deps — avoids re-registering the event listener on every theme change while still reading the latest theme value
