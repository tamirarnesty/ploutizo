## Key Changes

**AssigneeSection.tsx** — Replaced `!transaction` guard with `isInitialRender` ref. The effect now skips only the first render (preserving saved splits on mount) and fires on all subsequent `amountCents` changes in both create and edit mode. Added `useRef` to imports.

**TransactionForm.tsx** — Changed Notes+Tags wrapper from `flex gap-4` to `flex flex-col gap-4` (stacked). Removed `flex-1` from both Field wrappers. Increased Notes textarea from `rows={2}` to `rows={3}`.

**TransactionTagPicker.tsx** — Changed `filteredItems` from bare `filteredTagNames` to `createOption ? [...filteredTagNames, createOption] : filteredTagNames`. The Create item is now included in keyboard navigation.

**CategoriesSettings.tsx** — Removed `initialized` ref and its guard. The sync effect now runs whenever `categories` refetches (after mutations), so edits reflect immediately. Removed `useRef` from imports.
