## Security Threat Summary

| Threat | STRIDE | Mitigation |
|--------|--------|-----------|
| Invalid amountCents (negative, zero) | Tampering | Zod `z.number().positive()` on form + API validator |
| XSS via member/account name in DataGrid | Tampering | React text node rendering — no dangerouslySetInnerHTML |
| Negative balance pre-fill | Tampering | Pre-fill `Math.abs(balanceCents)` — amount field always ≥ 0 |
| notes field XSS or injection | Tampering | Form `maxLength={1000}` + Zod `.max(1000)` + server Zod validator; React JSX escaping |
