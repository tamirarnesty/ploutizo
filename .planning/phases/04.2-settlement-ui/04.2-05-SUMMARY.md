**2. [Rule 1 - Bug] ZodOptional.unwrap() for notes field validator**
- **Found during:** Task 1 (SettleDialog TypeScript check)
- **Issue:** `settleFormSchema.shape.notes` is `ZodOptional<ZodString>` — TanStack Form `AppField` typed as `string` rejects `ZodOptional` as onChange validator
- **Fix:** Used `settleFormSchema.shape.notes.unwrap()` to extract inner `ZodString`
- **Files modified:** `apps/web/src/components/dashboard/SettleDialog.tsx`
- **Commit:** `da3a953`

**3. [Rule 1 - Bug] Array.at(0) for undefined-safe first element**
- **Found during:** Task 1 (SettleDialog lint — no-unnecessary-condition rule)
- **Issue:** `account.members[0]` — TypeScript strict mode treats this as always truthy for typed arrays; lint rule flags the null guard as unnecessary condition
- **Fix:** `account.members.at(0) ?? null` — `.at()` returns `T | undefined` which makes the null coalesce meaningful
- **Files modified:** `apps/web/src/components/dashboard/SettleDialog.tsx`
- **Commit:** `da3a953`

**4. [Rule 3 - Blocking] Worktree path safety**
