## What Was Built

**Restore endpoint:** `PATCH /api/transactions/:id/restore` sets `deletedAt = null` for same-org deleted transactions. Returns 404 for wrong-org or already-active rows (idempotency guard via `isNotNull(deletedAt)` in WHERE clause).

**Expanded sort:** `GET /api/transactions?sort=type|category|account` now maps to `transactions.type`, `categories.name`, `accounts.name` columns respectively. No new joins needed — categories and accounts were already left-joined in `buildListQuery`. Unrecognized sort values fall back to `'date'`.

**CAD currency:** `formatCurrency` updated from `en-US`/`USD` to `en-CA`/`CAD`.
