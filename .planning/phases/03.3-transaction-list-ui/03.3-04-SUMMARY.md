Full /transactions page with URL-driven filters, server-side paginated+sorted DataGrid (9 columns), optimistic delete with 5s Undo toast, and Sidebar nav item.

## What Was Built

**Route (`_layout.transactions.tsx`):** TanStack Router `createFileRoute` with `validateSearch` using a Zod schema where every param uses `.catch()` (not `.default()`) to coerce invalid URL values to safe defaults. Exports `TransactionSearch` type used by both Transactions.tsx and TransactionsTable.tsx.

**Page shell (`Transactions.tsx`):**
- All 5 queries fired at top level: `useGetTransactions`, `useGetAccounts`, `useGetCategories`, `useGetOrgMembers`, `useGetTags` тАФ no waterfalls
- `buildCleanSearch` helper strips defaults before navigate calls
- `searchToFilters` / `filtersToSearch` round-trip URL params тЖФ `Filter<string>[]` for ReUI Filters state
- ReUI `Filters` component wired with 6 `FilterFieldConfig<string>` fields
- Disabled "Add transaction" button with `aria-disabled="true"` and tooltip

**DataGrid (`TransactionsTable.tsx`):**
- `useReactTable` with `manualPagination: true`, `manualSorting: true`, `rowCount: total`
- 9 column defs with correct `enableSorting` flags (date, type, category, account, amount = true; description, assignees, tags, actions = false)
- Per-type badge colors matching UI-SPEC.md (expense=destructive, income=emerald, transfer=secondary, settlement=blue, refund=orange, contribution=purple)
- Amount colors: expense/refund/settlement = text-destructive, income/contribution = emerald, transfer = muted
- `DynamicLucideIcon` with `Tag` fallback for category icons
- `AvatarGroup` with max 3 visible + overflow count for assignees
- Tag badges with tinted colour styling from `tag.colour`
- Delete: confirm Dialog тЖТ `deleteMutation.mutate(id)` тЖТ immediate `toast('Transaction deleted', { action: { label: 'Undo' } })`
- Two empty states: "No transactions yet" and "No transactions match your filters" with "Clear filters" link
- `motion-safe:animate-pulse` on all skeleton elements

**AppSidebar:** Transactions nav item added between Dashboard and Accounts with `Receipt` icon from lucide-react.
