# Summary: Create GitHub Actions CI workflow

## What Was Built

Created `.github/workflows/ci.yml` — a four-job parallel CI workflow that runs on every PR and push to main.

## Changes

### .github/workflows/ci.yml (new file)

Four concurrent jobs:
- **lint-typecheck** — runs `pnpm turbo lint` then `pnpm turbo typecheck`
- **test** — runs `pnpm turbo test`
- **format-check** — runs `pnpm turbo format:check` (added in 03.5-01)
- **build** — runs `pnpm turbo build`

Security/reliability features:
- `permissions: contents: read` — least-privilege, no write access for runners
- `concurrency: cancel-in-progress: true` — cancels stale runs on same PR branch
- `--frozen-lockfile` on all installs — fails if lockfile is out of date
- pnpm store cached via `actions/setup-node` with `cache: pnpm`
