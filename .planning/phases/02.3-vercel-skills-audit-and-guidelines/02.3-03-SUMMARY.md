### Task 2: Web Design Guidelines Audit — Violations Fixed

Fetched live guidelines from `https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md`.

**Violations found and fixed:**

#### LucideIconPicker.tsx
- `autoFocus` on search Input: added justification comment (Popover opened by user action — valid use case per guidelines)
- Placeholder `"Search icons..."` → `"Search icons…"` (typography: straight ellipsis → Unicode ellipsis)

#### AccountForm.tsx
- Missing `autoComplete` attributes on name input → `autoComplete="off"` + `name="account-name"` (custom account names should not be autocomplete candidates)
- Missing `autoComplete` on institution input → `autoComplete="organization"` + `name="account-institution"` (browser can suggest known financial institution names)
- Missing `autoComplete` on lastFour input → `autoComplete="off"` + `name="account-last-four"` (security-relevant partial card number)

#### CategoriesSettings.tsx
- `<span className="flex-1 text-sm">` for category names → `<span className="min-w-0 flex-1 truncate text-sm">` (flex child with potentially long text needs `min-w-0` to allow truncation)

#### CategoryForm.tsx
- `autoFocus` on category name Input: added justification comment (primary input in dialog opened by user action)
